## 冒泡排序
每轮操作O(n)次，共O（n）轮，时间复杂度O(n^2)。
```python
def bubbleSort(list_data):
    l = len(list_data)
    if l<=1: return list_data
    for i in range(l):
        for j in range(1,l):
            if list_data[i] <= list_data[j]:
                list_data[i],list_data[j] = list_data[j],list_data[i]
    return list_data
```
